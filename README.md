# PaperTarget Pro V3 Ultimate

> Global Research Publishing Intelligence Agent
> Version: 3.0 Ultimate

PaperTarget Pro V3 Ultimate is a **Codex Skill** — an AI Research Publishing Intelligence Agent that helps researchers maximize both acceptance probability and scientific impact through 20 integrated engine modules.

## Core Capabilities

- **Paper DNA Engine** — Automatic research hierarchy construction
- **Novelty Engine** — Innovation type and level diagnosis (L1-L5)
- **Scientific Value Engine** — Multi-dimensional value assessment
- **Publication Level Predictor** — Q1-Q4 / SCI / EI / Chinese core tier prediction
- **Journal Matching Engine** — Ten-layer matching system
- **Top Journal Recommender** — TOP 20 with tier groupings + category-specific lists
- **Editor Decision Simulator** — Desk reject / Accept / Revision probability simulation
- **Reviewer Brain Engine** — 3 simulated reviewers
- **Success Rate Engine** — Acceptance/rejection/revision rate percentages
- **Time Prediction Engine** — Full publication timeline estimation
- **Funding Match Engine** — NSFC / NSF / NASA / EU fund alignment
- **Academic Career Engine** — Graduation / Promotion / Talent plan pathway
- **Cover Letter Engine** — SCI/SSCI/EI/Chinese cover letters
- **Response to Reviewers Engine** — Bilingual point-by-point responses
- **Rejection Rescue Engine** — Rejection analysis + next journal recommendation
- **Academic Risk Control** — Predatory journal / early warning scoring
- **Trend Forecast Engine** — 3/5/10 year research trend analysis
- **Competitor Analysis Engine** — Competing papers/teams identification
- **AI Review Score** — Nature/Science/Cell/IEEE editorial simulation
- **Final Report** — Comprehensive Journal Submission Intelligence Report

## Usage

Provide paper details (title, abstract, keywords, or full text) along with academic goal, rejection history, or career stage. The agent activates relevant modules and produces a comprehensive report.

## Installation

Copy the papertarget-pro folder to Codex skills directory, or use the SKILL.md as a system prompt in ChatGPT, OpenWebUI, Coze, Claude Project, Gemini Gems, or any LLM platform.
